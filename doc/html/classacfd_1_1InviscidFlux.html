Detailed Description

Class to add contributions of inviscid fluxes to the residual.

Implements a Rhie-Chow stabilization in the mass flux, for which pressure reconstruction is needed.

See also
PressureReconstruction

vector<vector<double> > acfd::InviscidFlux::bvalues

Values of prescribed quantities corresponding to the flags in bcflags.

The value for each boundary consists of two numbers - this is needed for inflow velocity. For other types of boundary, we just read the first value corresponding to that boundary.
